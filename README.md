# Urban Digital Twins for Microclimate-Building-Energy Analysis

Static Mapbox GL JS site for Singapore building energy, 500 m district energy, and WRF-derived microclimate layers.
WRF weather layers support hourly playback through compact JSON time-series files under `data/weather-timeseries/`.

See `docs/mapbox-upload.md` for regenerating the weather and building-overview upload layers.
